# 🚀 Projects & Applications

### 📱 AroundFAST — Campus Accommodation Finder Platform
> **Cross-Platform Mobile Application** built to address campus real estate and accommodation search requirements.

* **Backend Infrastructure:** Built utilizing a Supabase serverless platform integrated with a PostgreSQL database.
* **Data Parsing Innovation:** Engineered a custom string-parsing filter to convert unstructured text entries (such as "5k to 8k") into queryable numbers for application range sliders.
* **Map Integration:** Implemented OpenStreetMap combined with asynchronous geocoding systems to map location pins without third-party API dependencies.
* **Project Links:** [GitHub Repository ↗](https://github.com/rayannkhn/AroundFast_v2) • [Production Deployment ↗](https://aroundfast.vercel.app/)

---

### 🌊 SAILAB — Flood Early Warning System
> **Real-time Telemetry Engine** developed to capture river data metrics and distribute automated emergency responses.

* **Data Pipeline:** Tracks real-time water discharge updates from Open-Meteo gauges, validating inputs against a multi-tier threat scale matrix.
* **Multi-Lingual Alerting:** Implemented an automated messaging and voice notification system using the Twilio API to broadcast warning scripts in English, Urdu, and Pashto.
* **Background Processing:** Programmed low-level background threads using `WorkManager` isolates to execute automated periodic checks and trigger notifications without keeping the application open.
* **Project Links:** [GitHub Repository ↗](https://github.com/rayannkhn/sailab-app) • [Production Deployment ↗](https://sailabpak.vercel.app/)

---

### 🖥️ Aura — AI-Powered Intelligent File Organizer
> **Machine Learning Desktop Application** designed to manage and optimize local directories.

* **Pipeline Structure:** Implemented a sequential pipeline executing K-Means clustering, KNN logic, $A^*$ path heuristics, Isolation Forest checks, and Linear Regression to cluster and clean storage space.
* **Feedback Loops:** Tracks file-management interactions to retrain local model elements and scale directory organization over time.
* **Project Links:** [GitHub Repository ↗](https://github.com/rayannkhn) • [Product Landing Page ↗](https://linuxfileorganizer.netlify.app/)

---

### 🛡️ Fake Financial Receipt Detection System
> **Computer Vision Forensic Pipeline** developed to validate and audit digital transaction printouts.

* **Core Features:** Combines localized pixel-level OpenCV image forensics, metadata pattern classifiers, and automated layout checks to identify forged records.
* **Project Repository:** [Access Project Infrastructure ↗](https://github.com/rayannkhn)

---

### 🧾 Receipt Zero — Paperless B2B Retail Platform
> **QR Digital Ledger System** designed for digital bill delivery and payment collection.

* **Hybrid Operations:** QR layouts support offline data caching alongside remote live tracking endpoints when networks are active.
* **Integrations:** Handles invoice delivery through the WhatsApp Business API, standard SMS, and payment processing loops.
